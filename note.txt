1. tạo 1 entities 

ví dụ entities User 
            
            import { Entity, Column, PrimaryGeneratedColumn } from 'typeorm';

            @Entity()
            export class User {
            @PrimaryGeneratedColumn()
            id: number;

            @Column()
            firstName: string;

            @Column()
            lastName: string;

            @Column({ default: true })
            isActive: boolean;
            }


sau khi tạo muốn nó tự generate ra thì phải gọi nó vô entities trong module
